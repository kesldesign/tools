This project focuses on developing parametric 3D organizers for jewelers, silversmiths, gemstone faceters, grinders, and hobby makers.

The project originated from the real needs of a jewelry workshop – the goal is to improve workspace organization, tool visibility, and overall work efficiency.


## 🎯 What the project addresses

The organizers are designed to store items such as:

- Milling cutters and rotary tools  
- Grinding and polishing wheels  
- Precious stones  
- Blanks and small components  

They help save space, minimize clutter, and keep the workspace systematic and organized.


## 🔧 Parametric modeling

Models are created in OpenSCAD as fully parametric:

- Easy dimension adjustments by changing a few values  
- Quick adaptation to specific tools  
- Source `.scad` files available for personal modifications  
- Ready-to-print module packages in `.stl` format  

The project is intended for both professionals and beginners.

## 🌍 Community and sharing

The purpose of the project is to share functional tools that the community can further develop.

Organizers can be modified, combined, and customized for specific operations.  
Every improvement, new variant, or feedback helps advance the project.

The goal is to create an open foundation where creators can build their own solutions without restrictions or unnecessary barriers.

#### Features

|Feature|Status|Description|
|-------|:---:|------|
|**Ready-to-print packages**|✔|Pre-prepared modules in .stl files for quick printing.|
|**Presets**|✔|OpenSCAD presets in .json format, allowing you to edit pre-configured modules.|
|**Antislip**|✔|Option to add anti-slip features to models (adhesive rubber pads 10mm). The model must be at least 40×40mm.|
|**Build plate check**|✔|Quick preview of model size relative to the 3D printer build plate. Printer list below.|
